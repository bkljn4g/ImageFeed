## Инструкция

Пользователю необходимо пройти регистрацию в сервисе Unsplash. Есть ограничение в 50 картинок, после чего нужно подождать час для последующего пользования.


## Технические требования

- Приложение поддерживает устройства iPhone с iOS 13 или выше, предусмотрен только портретный режим.
